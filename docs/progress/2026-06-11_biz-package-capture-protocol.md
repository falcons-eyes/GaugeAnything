## 로드맵 1단계 — 실측 mm GT 수집 체계 ✅ (도구 완성, 현장 수집 대기)

실측 mm GT는 모트 데이터이자 "1.53mm" 신뢰의 마지막 고리. 현장 촬영 전 단계를 전부 구축:

| 산출물 | 내용 | 검증 |
|---|---|---|
| [docs/CAPTURE_PROTOCOL.md](../CAPTURE_PROTOCOL.md) | 동일평면 배치·틸트 허용≤50°·캘리퍼 기록 규약·manifest 스키마·기여 라이선스(CC BY 4.0) | — |
| `experiments/make_print_board.py` | A4@300DPI 인쇄 보드 (ArUco 30mm×4 + 검증 눈금자) | **Spark 실행: 마커 4개 검출, 스케일 복원 오차 0.38%** ✓ |
| `experiments/real_mm_eval.py` | manifest → inspect_soft(PlaneScale) → MAE(mm)·±10%/20% 합격률 | 구문/인터페이스 검증 (실데이터 대기) |

남은 것: **물리적 촬영** (사용자/현장 채널 필요) — 보드 인쇄 → 결함 N개 촬영+캘리퍼 →
`captures/`에 넣고 `real_mm_eval.py` 실행. 예측: 합성 체인 오차(스케일 0.38% + e2e 5.6%)
기준 ±10% 합격 다수 기대, 실패 시 렌즈 왜곡 보정이 다음 과제.
