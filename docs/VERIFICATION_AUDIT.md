## 4. 정직한 약점 목록 (cherry-picking 위험 지점 — 숨기지 않음)

1. **krkCMd 이미지 레벨은 1개 이미지**(CMd_0.23_2mths/Image1, 6 stages × 90 profiles).
   다른 series/image로의 확장은 zip 선별 추출로 가능하나 미실행. → 일반화 주장 금지,
   "단일 시편 검증"으로 표기.
2. **E-mm-2b T-LESS n=53** (6 scenes × visib≥0.95 필터) — 작은 표본. 프롬프트 3종 일관성이
   보강 증거이나 n 확대 여지.
3. **동전은 동일 권종 LOO** — 절대 스케일 체인이 아니라 일관성 검증임을 항상 병기 (기존 표기 유지).
4. **profile CNN의 도메인** — 6400dpi 스캐너 콘크리트 한정 검증. 패키지 API에 명시.
5. **분할 벤치 시드 3개**는 데이터 서브샘플 시드이지 SAM3 추론 시드 아님 (추론은 결정적).
6. M2v2-a의 θ\* 선택셋은 train 서브샘플 300+val — 전체 train 미사용 (속도 타협, 명시됨).
