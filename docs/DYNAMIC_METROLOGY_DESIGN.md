## 2. 실험 사다리

| 실험 | 데이터 | 내용 | 산출 |
|---|---|---|---|
| **E-dyn-0** (스모크) | TUM checkerboard_large | 프레임별 체커보드 검출 → PlaneScale → 보드 셀/인접 구조 측정. **모션 강도(프레임간 포즈 변화율·블러 지표) vs 오차 곡선** | 핸드헬드 열화 곡선 1호 (CC BY — 논문 게재 클린) |
| **E-dyn-1** | EuRoC V1 easy→difficult | 60mm 체커보드 스케일 체인을 드론 블러 3단계에서 — 검출률·스케일 오차·측정 오차 | "marker chain의 동적 한계" 정량 |
| **E-dyn-2** | ARKitScenes 3DOD | prompt "chair/table/sofa" → SAM3 → depth(mm)/pose 스케일 → 3D 박스 치수 대비 | **인스펙터 워크스루 promptable 치수** (제품 시나리오 직결) |
| E-dyn-3 | ADT (가입 후) | 보행 에고센트릭 전 객체 치수 | 최강 GT — 사용자 가입 시 |
| E-dyn-K | KITTI | "round traffic sign" → 직경 → Ø600/750 스냅 검증 | known-object의 야외 동적판 |

공통 프로토콜: 시퀀스 단위 홀드아웃, 프레임 간 일관성(CV)·절대 오차·열화 곡선(블러/거리 bin),
게이트 실패율 보고 (정적 트랙과 동일 감사 문법).
